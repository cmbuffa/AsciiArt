# AsciiArt



Console application to print text in AsciiArt Format:



<pre>
# # ### #   #    #   #  ##  ##
# # #   #   #   # # # # # # # #
### ##  #   #   # # ### ##  ##
# # #   #   #   # # # # #   #
# # ### ### ###  #  # # #   #
</pre>
To run it, just clone and execute the following <pre>dotnet run</pre>
